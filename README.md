# Plataforma Saúde Mental

Bem-vindo ao repositório da **Plataforma Saúde Mental**! Este projeto foi desenvolvido para fornecer uma aplicação moderna e responsiva utilizando as seguintes tecnologias principais:

- **React** com **TypeScript**
- **Vite** para construção e desenvolvimento
- **Tailwind CSS** para estilização

## Funcionalidades

- Componentes reutilizáveis com `shadcn/ui`
- Suporte a temas e variantes de botões
- Configuração otimizada para desenvolvimento rápido

## Estrutura do Projeto

A estrutura principal do projeto é organizada da seguinte forma:

```
public/         # Arquivos estáticos
src/            # Código-fonte principal
  assets/       # Recursos como imagens e fontes
  components/   # Componentes reutilizáveis
  contexts/     # Contextos globais do React
  hooks/        # Hooks personalizados
  lib/          # Funções utilitárias
  pages/        # Páginas da aplicação
```

## Como Executar o Projeto

1. Clone o repositório:

   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```

2. Instale as dependências:

   ```bash
   pnpm install
   ```

3. Inicie o servidor de desenvolvimento:

   ```bash
   pnpm dev
   ```

4. Acesse a aplicação no navegador em: [http://localhost:5173](http://localhost:5173)

## Scripts Disponíveis

- `pnpm dev`: Inicia o servidor de desenvolvimento.
- `pnpm build`: Gera a versão de produção.
- `pnpm preview`: Visualiza a versão de produção localmente.
